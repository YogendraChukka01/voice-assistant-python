## Project Description

Voice Assistant for Desktop Automation is an educational Python project created by Yogendra Chukka. The assistant, named SREE, demonstrates early desktop automation with speech recognition, text-to-speech, browser automation, and keyboard automation.

## Features

- Speech recognition using microphone input.
- Text-to-speech responses.
- Opening YouTube and websites via voice commands.
- Simple Notepad typing automation.
- Password-protected launch.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/YogendraChukka01/voice-assistant-python.git
cd voice-assistant-python
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the assistant script:

```bash
python sree_assistant.py
```

Follow the voice and console prompts. The example password is stored as a placeholder in `sree_assistant.py`.

## Requirements
